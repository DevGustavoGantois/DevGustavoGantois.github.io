Minhas SoftSkills

HTML

HTML5 é a quinta versão da linguagem de marcação HTML (Hypertext Markup Language), usada para estruturar e exibir conteúdo na web. Introduz novos elementos, como header, footer, e nav, integração de APIs...

CSS

CSS (Cascading Style Sheets) é uma linguagem de estilo utilizada para controlar a apresentação e o layout de elementos HTML em páginas da web. Com o CSS, definindo cores, fontes, espaçamentos, bordas e outros estilos visuais para criar uma aparência consistente e atraente para o conteúdo web...

JavaScript

JavaScript é uma linguagem de programação de alto nível, conhecida por sua capacidade de tornar as páginas web interativas e dinâmicas. Ele é usado para adicionar comportamentos, manipular elementos HTML, responder a eventos do usuário e realizar operações assíncronas, como solicitações de rede (AJAX)...

REACT JS

React.js é um framework de JavaScript para construção de interfaces de usuário (UI) interativas e reativas em aplicações web. Baseado em componentes reutilizáveis e o conceito de Virtual DOM, React permite criar interfaces eficientes e escaláveis. Com uma vasta comunidade e um ecossistema de ferramentas robusto, React é amplamente adotado para o desenvolvimento de aplicações modernas e dinâmicas na web.
